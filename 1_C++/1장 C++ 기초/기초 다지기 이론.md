#기본적인 기초 다지기

[전처리 지시자 예시](예제1 기초 출력하기.cpp)
| 전처리 지시자 | 기눙 | 사용 예 |
| :-- | :-- | :-- |
| #include <[파일]> | 해당 '파일'의 함수를 불러온다. | 해당 파일의 함수를 사용할때 가져온다. |
| #define [키] [값] | 해당 키를 모두 지정한 값으로 지정한다. | 주로 상수값이나 매크로 정의 할때 사용한다. |
| #ifdef [키] <br>#endif <br><br>#ifndef <br>#endif | . | 해당 파일의 함수를 사용할때 가져온다. |
| #pragma [xzy] | 지정 '파일'의 함수를 불러온다. | 해당 파일의 함수를 사용할때 가져온다. |
