## 1.1 C++의 기초

C++은 C언어에서 클래스가 추가되고 불편하거나 어색한 부분을 개선한 언어이다.

### 1.1.1 기본 구조

```cpp
// 예제
import <iostream> // c++20 이전 버전에서는 #include <iostream>

int main()
{
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

```text
[출력]

Hello, World!

```

이 기본 예제에서 포함되어 있는 개념을 알아보자

- 주석
- 모듈 임포트
- main() 함수
- I/O 스트림

### 주석

주석은 프로그래머에게 전달하는 메시지나 기록을 뜻한다. 프로그램이 컴파일하는 과정에서는 주석을 무시하고 실행이 된다

```cpp
예시)

// [메시지]

/*
[메시지]
*/
```
